## 简介
### 技术栈
>  * Vue
>  * node.js
>  * mongodb
>  * Element-ui
### 改动
> 1. 使用Node.js创建了一个简单地接口服务器,提供接口和数据
> 2. 部分布局样式的改变

### Node服务器的启动
键入文件夹server中
执行 `npm install`,等待下载完毕后执行 `npm run start`
